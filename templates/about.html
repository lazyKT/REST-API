About

    Availble Routes

  • /index : Home Page
  • /about : Docs Page
  • */register : Register a new User
  • */login : Login
  • */logout : Logout
  • /confirm/token : User Confirmation Link via Email
  • */deactivate/username : Deactivate the account
  • */re-activate : Re-activate the account
  • /forget-password : Forget Password on Login
  • /reset-password/token : Reset-Password link via Email
  • */user/user_id : Get user details
  • */changepassword/user_id : Change User Password via Setting Page
  • */upload/avatar : Upload User Avatar
  • */avatar/user_id : Fetch Avatar related to User ID
  • */process : request Convertion to Mp3 file from video url
  • */mp3Convert/status/task_id : Check the mp3 Convertion Status with convertion_task_id
  • */listen/song_id : Listen or Play the mp3 file
          Routes with * means API Key is required for request.
